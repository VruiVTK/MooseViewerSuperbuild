Welcome to MooseViewerSuperbuild

This is the superbuild code that helps build MooseViewer along with all its pre-requisites including:

  • Vrui
  • VTK with the vtkRenderingExternal module turned ON

Clone MooseViewerSuperbuild source

MooseViewerSuperbuild can be built from source obtained using one of the download links on this page or using the commandline as follows:

$ git clone https://github.com/VruiVTK/MooseViewerSuperbuild.git

The above command creates a directory MooseViewerSuperbuild with the source code.

Configure Instructions

Configure the build using CMake keeping default options. Once configured, press 'Generate' to generate the makefile.

Build Instructions

Build using the make tool.

$ make

Run MooseViewer

Building the superbuild generates the application executable at <MooseViewerSuperbuild-build-dir>/MooseViewer/bld/ One can run the application by following the instructions provided here.
